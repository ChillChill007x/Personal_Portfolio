# 📖 Portfolio Setup Guide

## 🚀 Quick Start

### 1. Clone or Fork this Repository
```bash
git clone https://github.com/yourusername/your-portfolio.git
cd your-portfolio
```

### 2. Customize Your Profile
Edit `README.md` and replace:
- `[Your Name]` with your actual name
- `yourusername` with your GitHub username
- `Suphakit.f@kkumail.com` with your email

### 3. Add Your Content

#### For each lab folder:
1. Add your PDF reports
2. Add screenshots to `/images` folder
3. Add configuration files
4. Update the README in each folder

#### Directory Structure:
```
portfolio/
├── README.md (Main portfolio page)
├── lab1/
│   ├── README.md
│   ├── images/
│   └── configs/
├── lab2/
│   ├── README.md
│   └── videos/
├── lab3/
│   ├── README.md
│   └── LAB3.pdf
├── lab4/
│   └── README.md
├── new-network/
│   └── README.md
├── sprint-alpha/
│   └── README.md
└── project-network/
    └── README.md
```

### 4. Commit and Push
```bash
git add .
git commit -m "Initial portfolio setup"
git push origin main
```

### 5. Enable GitHub Pages (Optional)
1. Go to Settings > Pages
2. Select branch: `main`
3. Click Save
4. Your portfolio will be live at `https://yourusername.github.io/portfolio`

## 🎨 Customization Tips

### Add Badges
- Use [shields.io](https://shields.io) for custom badges
- Add technology badges for your skills

### Add GitHub Stats
- Customize the stats card theme in README.md
- Options: `default`, `dark`, `radical`, `merko`, `gruvbox`, etc.

### Add Screenshots
1. Create an `images` folder in each lab directory
2. Add screenshots: `![Description](./images/screenshot.png)`

### Add Videos
1. Upload to YouTube or Google Drive
2. Add links or embed: `[![Video](thumbnail.jpg)](video-link)`
